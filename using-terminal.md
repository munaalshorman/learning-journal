## Basic navigation

 1. *pwd command* It gives us the absolute path
 2. ls to know where we are. 
 ```
 ls [options][location]
 ```
 1. **Baths:**
 ```
  ~ (tilde) -  a shortcut for your home directory.
  ```
  ```
  . (dot) -  a reference to your current directory.
 ```
 1. *cd [location]* to go to directory
